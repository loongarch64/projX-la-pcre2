# projX-la-pcre2

## 项目名称

pcre2函数库的JIT代码生成。

## 支持单位

龙芯中科技术股份有限公司

## 项目描述

PCRE2库是一组C函数，使用与Perl 5相同的语法和语义实现正则表达式模式匹配。项目任务是添加在LoongArch平台上的即时编译(JIT)支持。
使用./configure --enable-jit 进行验证。编写必要的测试用例验证代码正确性。
